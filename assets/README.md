# Quotidian Reel Assets

Place your custom media files here for reel generation.

## Music

Place your background music track as `music.mp3` in this directory.

**Requirements:**
- Format: MP3
- Target loudness: ~-16 LUFS
- Style: Ambient, minimal, doesn't compete with quote
- Duration: Should be at least 8 seconds (reel is ~7.3s)

If no music file is found, the generator will create a simple procedural ambient pad as a placeholder.

## Audio Specifications

| Element | Level | Notes |
|---------|-------|-------|
| Music | ~-16 LUFS | Background |
| Letter SFX | -22 to -18 LUFS | Generated automatically |
| Sample rate | 48 kHz | For final output |
